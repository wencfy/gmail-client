# Gmail Client

> A desktop client for Gmail. Totally rendered by https://mail.google.com.

- Cross Platform
- Custom Styles
- Menu Bar Unread Badge for Timely Reply

## Developing

#### Install

```bash
yarn install
```

#### Run

```bash
yarn start
```

#### Build

```bash
yarn build:<macos|linux|windows>
```

The build output can be found in the dist/ folder.
